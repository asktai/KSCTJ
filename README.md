以上是"竹小苗口算出题机"的完整实现代码，这份代码实现了：
主要功能模块
1. 主题界面设计
清新的竹子主题配色（绿色系）配合飘落竹叶动画
圆角卡片式设计，使用明亮色彩（绿、橙、蓝）配合emoji图标
响应式布局，支持电脑、手机、平板，兼容鸿蒙、安卓、苹果系统
2. 选题功能
年级选择（1-6年级）与上下册多选
根据教学大纲自动匹配对应题型（20以内加减法到分数混合运算）
三级难度设置（初入江湖/小有成就/身手不凡）
题目数量（0-100）和数字范围（0-500）自定义
负数开关（新手保护模式/高手过招）
3. 出题与答题功能
多题模式（单列/双列/三列/四列）和单题模式
支持键盘方向键导航和Enter键提交
加减块和键盘双输入，实时同步
智能检查答案（绿色✓/红色✗）并显示正确答案
4. 计时与成绩系统
实时答题计时（分秒显示）
自动计算得分（正确数/总数 百分制）
分级显示成绩颜色（100分绿/80分以上蓝/60分以上黄/60分以下红）
根据正确率显示不同emoji鼓励表情（🏆😊👍🌱💪）
全对时触发庆祝动画（五彩纸屑效果）
5. 打印与分享功能
打印优化：自动隐藏按钮、调整为双列布局、防跨页处理
移动设备打印提示（建议导出PDF或截屏）
分享功能支持微信、钉钉，自动复制分享文本
文件名规则：姓名+主题+时间（精确到秒）
6. 特色功能
宣传框1：随机滚动显示励志名言（20秒周期，随机颜色）
宣传框2：三栏布局，左右预留二维码位
作者链接和更新下载链接（提取码zyxx）
重置功能恢复所有默认值
技术亮点
纯前端实现：无需后端，单HTML文件即可运行
响应式设计：CSS Grid + Flexbox适配各种屏幕
打印优化：使用@media print和break-inside: avoid确保打印效果
性能优化：CSS动画使用GPU加速，避免重排重绘
无障碍支持：大按钮设计，适合儿童手指操作，立体按压反馈
