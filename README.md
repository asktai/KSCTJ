三、主要功能使用流程
【基础使用流程】
1️⃣ 选择题型（二选一）
方式A：切换"大纲题型"标签，选择年级→册别→具体题型
方式B（推荐）：在"自定义题型"中展开分类，点击选择题型（可多选）
2️⃣ 设置参数
调整题目数量（默认10题）
设置数值范围（默认20以内）
选择难度和计算范围
3️⃣ 生成题目
点击 "🚀出题" 按钮，系统自动生成题目并显示在下方答题区
4️⃣ 答题方式
多题模式：直接在每题后的输入框填写答案
单题模式：在当前题目输入框答题，使用"上一题/下一题"切换或键盘方向键
5️⃣ 提交与批改
快速批改：点击 "✅提交" 按钮，仅显示对错标记
详细批改：点击 "📝批改" 按钮，显示对错及正确答案
批改后自动显示成绩统计和用时
6️⃣ 保存与分享
🖨️ 打印PDF：生成PDF格式练习题（带页眉信息）
💾 存图：将当前题目保存为PNG图片（文件名自动包含时间戳）
💬 微信分享 / 📱钉钉分享：复制分享文本到剪贴板

七、注意事项
⚠️ 使用建议
数值范围：设置过大可能导致计算困难，建议根据年级选择合适范围
负数模式：带余数除法暂不支持负数，会自动切换为非负模式
题目数量：建议一次不超过50题，过多可能影响生成速度
竖式/脱式：属于长题型，建议选择单列或双列显示

学生信息：自动保存在浏览器本地，下次打开自动填充
计时数据：累计计时在页面刷新后保留，关闭浏览器后清除
历史记录：V2.0.4后已移除自动历史记录功能，需手动保存

八、常见问题
Q：如何清空已填写的答案重做？
A：点击"🔄 重做"按钮，清空所有答案和批改标记，计时器重置，可重新答题。
Q：分数题如何输入答案？
A：分数线模式下分别输入分子和分母；斜线模式下输入格式如 3/4。
Q：为什么有些题型无法生成题目？
A：可能是数值范围设置过小，导致无法满足题目要求（如需要进位的加法但范围太小）。建议增大数值范围或降低难度。
Q：如何保存我的练习记录？
A：使用"💾 存图"功能保存为图片，或使用浏览器打印功能保存为PDF。
提示：本工具完全离线可用，数据仅保存在本地浏览器中，不会上传到任何服务器，保护学生隐私。

🔢 基础运算类
纯加法：两个数相加
纯减法：两个数相减（支持退位）
连加法：三个数连续相加
连减法：三个数连续相减
✖️ 乘除运算类
乘法：表内乘法及扩展
除法：整除运算
带余数的除法：有余数除法（答案格式：商...余数）
🔀 混合运算类
加减混合：随机生成加法或减法
两数/三数加减混合：多步加减组合
带括号算式：包含小括号的优先级运算
填空题形式：等号左侧或右侧留空填空
四则混合运算：加减乘除综合（先乘除后加减）
🧮 竖式与脱式类
竖式计算：竖排数字格式，适合列竖式练习
脱式计算：多步混合运算，需写中间步骤
加法进位/减法退位：专项进位借位训练
🍕 分数运算类
分数加法/减法：同分母、异分母分数相加减（自动通分）
分数乘法：分子乘分子、分母乘分母（自动约分）
分数除法：乘以倒数（自动约分）
多选功能：
勾选"允许多选"后，可同时选择多种题型，系统会随机混合出题
适合综合性练习，模拟真实考试场景
3.2 题目参数设置
题目数量设置：
范围：1-100题
调节方式：点击"+" "-"按钮，或直接输入数字
建议：日常练习建议10-20题，测试建议30-50题
数值范围设置：
范围：1-10000
作用：控制题目中出现的数字大小及答案范围
智能限制：选择"带余数的除法"时，若范围小于10会自动调整为10
难度等级：
🌱 初入江湖：数字较小，进位借位简单，适合基础练习
🌿 小有成就：中等复杂度，适合巩固提升
🎋 身手不凡：大数字、多步骤、复杂混合运算，适合挑战
计算范围：
🛡️ 非负模式：确保所有题目结果不为负数（适合低年级）
⚔️ 允许负数：开启负数运算（适合高年级）
3.3 出题与答题
出题模式选择：
📋 多题模式（默认）：
一次显示所有题目，类似传统试卷
支持设置列数：1列（竖版）、2列（默认）、3列、4列
适合：打印纸质练习、长时间测试
📝 单题模式：
一次只显示一道题，逐题作答
显示进度条和题号计数（如：第5/20题）
支持键盘方向键切换（←上一题，→下一题）
适合：专注训练、计时挑战、低龄学生
分数显示模式（针对分数题型）：
分数线模式：竖式分数显示，分子分母上下排列，美观正式，适合打印
斜线模式：横向显示（如3/4），输入方便，适合在线快速答题

4.2.1 竖式与脱式计算

题目以竖排格式显示，数字对齐

4.2.2 分数运算
输入方法：
分数线模式：分别输入分子（上框）和分母（下框）
斜线模式：在输入框填写"分子/分母"（如输入"3/4"）
自动约分：
系统会自动将答案约分到最简形式
如输入"6/8"，系统认可但正确答案显示"3/4"
建议学生也练习约分，填写最简分数
通分处理：
异分母加减法会自动通分计算
学生只需填写最终结果，无需展示通分过程（适合填空）
4.2.3 单题模式与多题模式切换
切换时机：必须在出题前选择模式，出题后不可切换
单题模式优势：
减少视觉干扰，专注力训练
配合计时器使用，适合速算挑战
防止偷看后面的题目
多题模式优势：
整体把握，适合模拟考试
便于打印纸质版
可自由跳转答题顺序

第七章 常见问题解答（FAQ）
7.1 题型相关
Q1：为什么选择了"带余数的除法"却无法生成题目？
A：请检查数值范围是否设置过小（如小于10）。带余数除法需要被除数大于除数且有余数，若范围太小无法满足条件，建议将数值范围设为20以上。
Q2：分数运算的答案格式是什么？
A：支持两种格式：分数线模式分别输入分子分母；斜线模式输入"分子/分母"（如3/4）。系统会自动约分，填写6/8也会被判正确，但建议练习时填写最简分数。
Q3：竖式计算如何输入答案？
A：竖式题在屏幕显示为竖排格式，但答案只需在横线后的输入框填写最终数字即可，无需画出竖式。
Q4：为什么有些年级没有下册内容？
A：部分高年级（如六年级下册）内容可能因教学安排不同而较少，建议选择"自定义题型"补充练习。
7.2 操作相关
Q5：如何清空已填写的答案重新做题？
A：点击"🔄 重做"按钮，会保留当前题目但清空所有答案和批改标记，计时器重置，可重新答题。如需更换题目，需点击"🚀出题"重新生成。
Q6：学生信息会自动保存吗？
A：是的。学校、班级、姓名会自动保存在浏览器本地，下次打开自动填充。但若清理浏览器缓存或更换设备，信息会丢失。
Q7：如何调整计时器位置？
A：鼠标按住计时器窗口（避开按钮区域）即可拖动到屏幕任意位置，防止遮挡题目。位置在刷新后会重置。
Q8：打印时为什么显示不完整？
A：请检查浏览器打印设置：1) 纸张大小设为A4；2) 边距设为默认；3) 取消勾选"页眉页脚"（浏览器默认的网址显示）；4) 开启"背景图形"以显示颜色。
7.3 技术问题
Q9：软件可以在没有网络的情况下使用吗？
A：可以。下载HTML文件到本地后，双击即可在浏览器中打开，完全离线可用。但"分享"功能需要网络访问在线版链接。
Q10：为什么二维码显示"不可用"？
A：可能是网络问题导致QRCode库加载失败，或浏览器安全设置阻止了二维码生成。这不影响核心功能，可刷新页面重试，或直接访问在线版链接。
Q11：保存图片时出现错误怎么办？
A：可能是题目过多导致内存不足。建议：1) 减少题目数量至30题以内；2) 使用系统截图功能替代；3) 检查浏览器是否允许下载文件。
Q12：如何更新到最新版本？
A：点击信息面板中的"更新升级（提取码:zyxx）"链接，或访问在线版自动获取最新版本。本地文件需手动下载新版HTML替换。
